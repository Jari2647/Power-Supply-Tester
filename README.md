# Power-Supply-Tester
Custom ATX power supply tester built on Arduino Mega 2560

Hardware Components
  - Arduino Mega 2560
  - 3 IRLZ44N MOSFETs
  - ATX Breakout Board
  - 1602A LCD Display
  - 2 Pushbuttons
  - 6x 10W, 10ohm ceramic power resistors
  - 2x 25, 1ohm ceramic power resistors
  - Various small resistors
  - Potentiometer

Library used - Integrated arduino 1602a LCD library
